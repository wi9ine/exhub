# 주문 리워드 종목 정보 조회

진행중인 주문 리워드 프로그램에 대한 정보 조회

## Request Header

| 필드                  | 필수   | 설명                                                       |
| :------------------ | :--- | :------------------------------------------------------- |
| X-COINONE-PAYLOAD   | true | Request body object -> JSON string -> base64             |
| X-COINONE-SIGNATURE | true | HMAC(X-COINONE-PAYLOAD, SECRET\_KEY, SHA512).hexdigest() |

## Response Body

| 필드                       | 타입             | 설명                                      |
| :----------------------- | :------------- | :-------------------------------------- |
| result                   | String         | 정상 처리 시 "success", 에러 발생 시 "error" 반환   |
| error\_code              | String         | 정상 처리 시 "0", 에러 발생 시 에러코드 "0" 이 아닌 값 반환 |
| programs                 | Array\[Object] | 배열 형태의 진행중인 주문 리워드 정보 목록                |
| - quote\_currency        | String         | 마켓 기준 통화                                |
| - target\_currency       | String         | 주문 통화                                   |
| - reward\_amount         | String         | 대상 리워드 금액                               |
| - contribution\_ratio    | String         | 기여도                                     |
| - expect\_reward\_amount | String         | 예상 리워드 금액                               |
| - score                  | String         | 내 점수                                    |
| - total\_score           | String         | 전체 점수                                   |
| - is\_maintenance        | String         | 점검 여부                                   |
| - competition\_level     | String         | 경쟁도 (LOW, MIDDLE, HIGH)                 |
