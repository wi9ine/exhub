# API 거래 수수료율 안내

코인원 Open API를 활용한 거래의 수수료율을 안내드립니다.

개인용 API로 발급한 키를 통한 주문에만 해당 수수료율이 적용되며 코인원 계정과 연동된 외부 서비스는 기본 수수료율이 적용됩니다.

# 거래 수수료 안내

코인원 Open API를 이용한 거래의 수수료율이 인하되어 아래의 수수료율이 적용됩니다.

* Open API 수수료 인하 : Maker/Taker 0.02% (2025년 1월 2일 변경 적용, [관련 공지](https://coinone.co.kr/info/notice/3649))
* 테더(USDT) 거래 수수료 인하 이벤트: Maker 0%/Taker 0.02% (별도 안내 시까지, [관련 공지](https://coinone.co.kr/info/notice/3688))
* 유에스디에스(USDS) 거래 수수료 인하 이벤트: Maker 0%/Taker 0.02% (별도 안내 시까지, [관련 공지](https://coinone.co.kr/info/notice/4261))

<br />

**유의사항**

* 신규 수수료율 적용 이전에 생성된 주문의 경우 기존 수수료율이 적용되오니 주의를 부탁드립니다.
* Open API 수수료 인하/테더(USDT) 수수료 인하 정책은 별도 종료 공지 시까지 운영되며, 조건이 변경될 수 있음을 양해 부탁드립니다.
