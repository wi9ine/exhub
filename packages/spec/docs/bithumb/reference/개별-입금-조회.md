# 개별 입금 조회

입금 UUID로 해당 입금 건의 입금 내역을 조회합니다.

> 예시코드는 JavaScript, Python, JAVA에 한해서만 제공합니다.

<br />

## **Request Parameters**

| 필드          | 설명                 | 타입     |
| :---------- | :----------------- | :----- |
| currency \* | 화폐를 의미하는 영문 대문자 코드 | String |
| uuid        | 입금 UUID            | String |
| txid        | 입금 TXID            | String |
