# 입금 주소 생성 요청

입금 주소 생성을 요청합니다.

> 예시코드는 JavaScript, Python, JAVA에 한해서만 제공합니다.

<br />

## **Request Parameters**

| 필드           | 설명                         | 타입     |
| :----------- | :------------------------- | :----- |
| currency \*  | 화폐를 의미하는 영문 대문자 코드(ex.BTC) | String |
| net\_type \* | 입금 네트워크(ex.BTC, DASH)      | String |

## **Response**

| 필드                 | 설명                 | 타입     |
| :----------------- | :----------------- | :----- |
| currency           | 화폐를 의미하는 영문 대문자 코드 | String |
| net\_type          | 입금 네트워크            | String |
| deposit\_address   | 입금 주소              | String |
| secondary\_address | 2차 입금 주소           | String |
