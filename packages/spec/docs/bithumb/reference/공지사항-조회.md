# 공지사항 조회

빗썸 공지사항을 조회합니다.

> 예시코드는 JavaScript, Python, JAVA에 한해서만 제공합니다.

<br />

## **Request Parameters**

| 필드    | 설명                                      | 타입     |
| :---- | :-------------------------------------- | :----- |
| count | 공지사항 게시물 조회 개수(기본 : 5, 최소 : 1, 최대 : 20) | Number |

> 🚧 안내사항
>
> * <span style={{ fontSize: "13px" }}> 현재 API 호출 제한은 <span style={{color: "#dd1e2e"}}><strong>IP당 초당 1회</strong></span>로 설정되어 있습니다. 다만, 시스템 상황이나 트래픽에 따라 이 제한은 변경될 수 있습니다.</span>

## **Response**

| 필드            | 설명                              | 타입             |
| :------------ | :------------------------------ | :------------- |
| categories    | 공지사항 카테고리                       | Array\[String] |
| title         | 공지사항 제목                         | String         |
| pc\_url       | 공지사항 URL                        | String         |
| published\_at | 공지사항 게시시간 (yyyy-MM-dd hh:mm:ss) | String         |
| modified\_at  | 공지사항 수정시간 (yyyy-MM-dd hh:mm:ss) | String         |

<br />

> 📘 Information
>
> * 공지사항 목록은 게시 시간(published\_at)을 기준으로 최신 순으로 제공됩니다.
