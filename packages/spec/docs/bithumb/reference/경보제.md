# 경보제

경보중인 마켓-코인 목록 조회

> 예시코드는 JavaScript, Python, JAVA에 한해서만 제공합니다.

<br />

## **Response**

<Table align={["left","left","left"]}>
  <thead>
    <tr>
      <th>
        필드
      </th>

      <th>
        설명
      </th>

      <th>
        타입
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        market
      </td>

      <td>
        빗썸에서 제공중인 시장 정보
      </td>

      <td>
        String
      </td>
    </tr>

    <tr>
      <td>
        warning\_type
      </td>

      <td>
        경보 유형\
        `- PRICE_SUDDEN_FLUCTUATION : 가격 급등락`\
        `- PRICE_DIFFERENCE_HIGH : 글로벌 시세차이`\
        `- SPECIFIC_ACCOUNT_HIGH_TRANSACTION : 소수계정 거래 집중`\
        `- TRADING_VOLUME_SUDDEN_FLUCTUATION : 거래량 급등`\
        `- DEPOSIT_AMOUNT_SUDDEN_FLUCTUATION : 입금량 급등`
      </td>

      <td>
        String
      </td>
    </tr>

    <tr>
      <td>
        warning\_step
      </td>

      <td>
        경보 단계\
        `- CAUTION : 주의`\
        `- WARNING : 경고`\
        `- DANGER : 위험`
      </td>

      <td>
        String
      </td>
    </tr>

    <tr>
      <td>
        end\_date
      </td>

      <td>
        가상 자산 경보 종료일시(KST 기준)\
        포맷: `yyyy-MM-dd HH:mm:ss`
      </td>

      <td>
        String
      </td>
    </tr>
  </tbody>
</Table>
