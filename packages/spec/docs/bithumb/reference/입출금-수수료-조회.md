# 입출금 수수료 조회

빗썸에 상장된 가상자산의 최소 입출금 수량과 입출금 수수료를 제공합니다.

> 예시코드는 JavaScript, Python, JAVA, Shell에 한해서만 제공합니다.

<br />

<br />

## **Request Parameters**

<Table align={["left","left","left"]}>
  <thead>
    <tr>
      <th>
        필드
      </th>

      <th>
        설명
      </th>

      <th>
        타입
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        currency
      </td>

      <td>
        화폐를 의미하는 영문 대문자 코드
        (ALL로 요청 시 전체 응답)
      </td>

      <td>
        String
      </td>
    </tr>
  </tbody>
</Table>

<br />

## **Response**

| 필드                                   | 설명                                      | 타입             |
| :----------------------------------- | :-------------------------------------- | :------------- |
| name                                 | 코인명                                     | String         |
| currency                             | 코인 심볼                                   | String         |
| networks                             | 입출금 네트워크 정보                             | Array\[Object] |
| networks.net\_name                   | 입출금 네트워크명                               | String         |
| networks.deposit\_fee\_quantity      | 입금 수수료                                  | NumberString   |
| networks.deposit\_minimum\_quantity  | 최소 입금 수량 (최소 입금 수량 미만으로 입금할 경우, 수수료 부과) | NumberString   |
| networks.withdraw\_fee\_quantity     | 출금 수수료                                  | NumberString   |
| networks.withdraw\_minimum\_quantity | 최소 출금 수량                                | NumberString   |
