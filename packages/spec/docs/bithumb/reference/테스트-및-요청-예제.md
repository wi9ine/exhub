# 테스트 및 요청 예제

WebSocket을 이용한 시세 수신 테스트를 진행할 경우 telsocket 혹은 wscat을 사용할 수 있습니다.

> wscat 을 이용하는 경우

```shell Shell
$ npm install -g wscat
$ wscat -c wss://ws-api.bithumb.com/websocket/v1
connected!
```

<br />

> telsocket 을 이용하는 경우

```shell Shell
$ telsocket -url wss://ws-api.bithumb.com/websocket/v1  
connected!
```
