# API 요청 수 제한 안내

안정적인 서비스 제공을 위해 API 유형별로 초당 요청 수를 제한합니다.

## 요청 제한 기준

| 구분              | 초당 최대 요청 수 |
| --------------- | :--------: |
| **Public API**  |    150회    |
| **Private API** |    140회    |

<Callout icon="⚠️" theme="warning">
  **주문 관련 API 추가 제한** — 주문 생성, 취소 등 주문(order) 관련 요청은 **초당 10회**를 초과할 경우 제한될 수 있습니다.
</Callout>

## 제한 초과 시 동작

요청 수 제한을 초과하면 API 사용이 일시적으로 제한됩니다.

* 제한이 해제될 때까지 잠시 대기한 후 재요청하세요.

<Callout icon="💡" theme="info">
  요청 간격을 일정하게 분산하면 제한에 걸릴 가능성을 줄일 수 있습니다. 한 번에 대량의 요청을 보내는 대신, 일정한 간격으로 호출하는 것을 권장합니다.
</Callout>

## 유의사항

* 사용자 급증 등 과도한 트래픽이 발생할 경우, 안정적인 서비스 제공을 위해 **별도의 공지 없이** API 요청량을 제한하거나 하향 조정할 수 있습니다.
* 초과 요청으로 API 사용이 제한된 후 자동으로 해제처리가 되지 않는 경우, **고객센터(1661-5566)** 로 문의하여 주시기 바랍니다.
