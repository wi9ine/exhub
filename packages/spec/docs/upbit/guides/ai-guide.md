# AI와 함께하는 업비트 API 연동

AI로 업비트 개발자 센터 문서를 효율적으로 탐색하고 개발에 활용해보세요.

## 시작하기

이제 업비트 개발자 센터의 문서를 AI 도구나 LLM 기반 IDE에 연동하여, 필요한 API를 빠르게 찾고 연동 코드를 자동으로 생성할 수 있습니다. 본 문서에서는 업비트 API 문서를 AI 도구에 효과적으로 연결해주는 llms.txt 사용 방법과 AI 활용 사례를 소개합니다.

<br />

## llms.txt란?

llms.txt는 ChatGPT, Claude와 같은 LLM이 문서를 효과적으로 탐색하고 이해할 수 있도록 돕는 표준 포맷입니다. AI는 이 파일을 기반으로 전체 문서 구조를 파악하고, 각 문서의 .md 형태 URL(문서 URL 뒤에 .md를 붙인 형태)에 직접 접근해 Markdown 형식의 본문을 직접 읽고 처리할 수 있습니다. 아래의 llms.txt 링크를 AI 도구에 컨텍스트로 전달해, 보다 정확하고 편리한 문서 탐색과 개발 과정을 경험해보세요.

## AI 시대의 투자, 업비트 개발자 센터에서 시작하세요.

업비트 개발자 센터는 AI를 활용한 보다 편리한 개발 및 업비트 API 연동을 지원하기 위해, 다양한 AI 친화 기능들을 지속적으로 확대해 나갈 예정입니다. 앞으로 더욱 매끄럽고 흥미로운 투자 전략 구현 경험을 제공할 업비트 개발자 센터의 변화에 많은 관심 부탁드립니다.
