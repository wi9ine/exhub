# 개인지갑 주소 등록

개인 지갑 주소를 업비트의 출금 허용 주소로 등록하는 방법을 안내합니다.

# 개인 지갑 주소 등록

업비트에서 지원하는 개인지갑은 [이용자 가이드](https://support.upbit.com/hc/ko/articles/5077581709209-%EC%97%85%EB%B9%84%ED%8A%B8%EC%97%90%EC%84%9C-%EC%A7%80%EC%9B%90%ED%95%98%EB%8A%94-%EA%B0%9C%EC%9D%B8%EC%A7%80%EA%B0%91%EC%97%90%EB%8A%94-%EC%96%B4%EB%96%A4-%EC%A2%85%EB%A5%98%EA%B0%80-%EC%9E%88%EB%82%98%EC%9A%94)에서 확인 가능합니다.

개인지갑을 출금 허용 주소로 등록하려면, 먼저 해당 지갑이 주소 관리 메뉴에서 등록되어 있어야 합니다.

### 개인지갑 주소 관리

# 출금허용주소 등록

업비트 PC 웹에 로그인한 후, \[마이페이지 > [Open API 관리](https://upbit.com/mypage/open_api_management)]에서 \[디지털 자산 출금주소 관리 탭]으로 이동합니다. 이동 후 \[출금허용주소 등록하기 → 주소 추가] 버튼을 클릭합니다.

# 인증 및 출금 주소 등록 완료

주소 등록 페이지에서 \[확인] 버튼을 클릭한 후, ‘인증 대기’ 화면에서 인증 대기 중인 주소를 확인할 수 있습니다.\
오른쪽 상단의 \[2채널 인증하기] 버튼을 클릭해 인증할 항목을 선택해 주세요. 인증이 완료되면 ‘디지털 자산 출금 주소 등록 성공’ 팝업이 표시됩니다.

# 출금허용주소 삭제

주소 관리는 [디지털 자산 출금 주소 관리](https://upbit.com/mypage/open_api_management?tab=fund_source) 메뉴에서 가능합니다.\
삭제하려는 주소를 선택한 후, 좌측 상단의 \[삭제] 버튼을 클릭하고 2채널 인증을 완료하면, 해당 출금 주소가 삭제 처리됩니다.
