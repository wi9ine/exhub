# 설치 및 설정

## 설치

```bash npm2yarn
npm install -g @exhub/mcp
```

또는 `npx`로 설치 없이 바로 실행할 수 있습니다:

```bash
npx @exhub/mcp --exchange upbit
```

## 실행

```bash
exhub-mcp --exchange <거래소>
```

`--exchange` 옵션에 사용할 거래소를 지정합니다:

```bash
exhub-mcp --exchange upbit
exhub-mcp --exchange bithumb
exhub-mcp --exchange coinone
exhub-mcp --exchange gopax
exhub-mcp --exchange korbit
```

import { Callout } from "nextra/components";

## 환경 변수

### 공통 옵션

모든 거래소에서 사용할 수 있는 공통 환경 변수입니다:

| 환경 변수 | 설명 | 기본값 |
|----------|------|--------|
| `EXHUB_<EXCHANGE>_BASE_URL` | API 기본 URL | 각 거래소 공식 URL |
| `EXHUB_<EXCHANGE>_TIMEOUT_MS` | 요청 타임아웃 (밀리초) | `10000` |

예: `EXHUB_UPBIT_BASE_URL`, `EXHUB_BITHUMB_TIMEOUT_MS`

### 거래소별 인증 키

Private API 도구를 사용하려면 해당 거래소의 API 키를 환경 변수로 설정해야 합니다.

| 거래소 | 환경 변수 |
|--------|----------|
| **Upbit** | `EXHUB_UPBIT_ACCESS_KEY`, `EXHUB_UPBIT_SECRET_KEY` |
| **Bithumb** | `EXHUB_BITHUMB_API_KEY`, `EXHUB_BITHUMB_SECRET_KEY` |
| **Coinone** | `EXHUB_COINONE_ACCESS_TOKEN`, `EXHUB_COINONE_SECRET_KEY` |
| **Gopax** | `EXHUB_GOPAX_API_KEY`, `EXHUB_GOPAX_SECRET_KEY` |
| **Korbit** | `EXHUB_KORBIT_API_KEY`, `EXHUB_KORBIT_SECRET_KEY` |

<Callout type="info">
  Public API(시세, 호가, 체결 내역 등)는 인증 없이 사용할 수 있습니다. API 키를 설정하지 않아도 Public API 도구는 정상 동작합니다.
</Callout>
