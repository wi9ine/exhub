# MCP 개요

## MCP란?

[MCP(Model Context Protocol)](https://modelcontextprotocol.io/)는 AI 모델이 외부 도구와 데이터에 접근할 수 있게 해주는 개방형 프로토콜입니다. MCP 서버를 등록하면 AI가 해당 서버의 도구를 직접 호출할 수 있습니다.

## ExHub MCP

ExHub MCP 서버(`@exhub/mcp`)는 ExHub SDK의 모든 기능을 AI 도구로 노출합니다. 코드를 작성하지 않고도 자연어로 거래소 데이터를 조회하고 활용할 수 있습니다.

### 사용 예시

AI 클라이언트에 ExHub MCP를 등록하면 이런 대화가 가능합니다:

- "비트코인 현재가 알려줘"
- "고팍스에서 최근 BTC 체결 내역 보여줘"
- "이더리움 호가 조회해줘"

### 지원 거래소

| 거래소  | Exchange 키 |
| ------- | ----------- |
| Upbit   | `upbit`     |
| Bithumb | `bithumb`   |
| Coinone | `coinone`   |
| Gopax   | `gopax`     |
| Korbit  | `korbit`    |
