# 클라이언트 설정

사용하는 AI 클라이언트에 맞는 설정 방법을 확인하세요.

    ## Claude Desktop

    설정 파일을 열어 MCP 서버를 추가합니다.

    **설정 파일 위치:**
    - macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
    - Windows: `%APPDATA%\Claude\claude_desktop_config.json`

    ```json
    {
      "mcpServers": {
        "exhub-upbit": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "upbit"],
          "env": {
            "EXHUB_UPBIT_ACCESS_KEY": "your-access-key",
            "EXHUB_UPBIT_SECRET_KEY": "your-secret-key"
          }
        }
      }
    }
    ```

    설정 후 Claude Desktop을 재시작하면 적용됩니다.

    ### 여러 거래소 동시 사용

    ```json
    {
      "mcpServers": {
        "exhub-upbit": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "upbit"]
        },
        "exhub-bithumb": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "bithumb"]
        }
      }
    }
    ```

    ## Claude Code

    CLI 명령으로 MCP 서버를 등록합니다.

    ```bash
    # 현재 사용자 전역 등록
    claude mcp add exhub-upbit --scope user -- npx @exhub/mcp --exchange upbit

    # 현재 프로젝트에만 등록
    claude mcp add exhub-upbit --scope project -- npx @exhub/mcp --exchange upbit
    ```

    ### 환경 변수 설정

    Private API를 사용하려면 환경 변수와 함께 등록합니다:

    ```bash
    claude mcp add exhub-upbit --scope user \
      -e EXHUB_UPBIT_ACCESS_KEY=your-access-key \
      -e EXHUB_UPBIT_SECRET_KEY=your-secret-key \
      -- npx @exhub/mcp --exchange upbit
    ```

    ### 등록 확인

    ```bash
    claude mcp list
    ```

    ## Cursor

    MCP 설정 파일에 서버를 추가합니다.

    **설정 파일 위치:**
    - 글로벌: `~/.cursor/mcp.json`
    - 프로젝트: `.cursor/mcp.json` (프로젝트 루트)

    ```json
    {
      "mcpServers": {
        "exhub-upbit": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "upbit"],
          "env": {
            "EXHUB_UPBIT_ACCESS_KEY": "your-access-key",
            "EXHUB_UPBIT_SECRET_KEY": "your-secret-key"
          }
        }
      }
    }
    ```

    설정 후 Cursor를 재시작하면 적용됩니다.

  </Tabs.Tab>

    ## VS Code

    프로젝트의 `.vscode/mcp.json` 파일에 서버를 추가합니다.

    ```json
    {
      "servers": {
        "exhub-upbit": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "upbit"],
          "env": {
            "EXHUB_UPBIT_ACCESS_KEY": "your-access-key",
            "EXHUB_UPBIT_SECRET_KEY": "your-secret-key"
          }
        }
      }
    }
    ```

    또는 `settings.json`에 추가할 수도 있습니다:

    ```json
    {
      "mcp": {
        "servers": {
          "exhub-upbit": {
            "command": "npx",

    ## Zed

    Zed 설정 파일에 context server를 추가합니다.

    **설정 파일 위치:** `~/.config/zed/settings.json`

            "args": ["@exhub/mcp", "--exchange", "upbit"],
            "env": {
              "EXHUB_UPBIT_ACCESS_KEY": "your-access-key",
              "EXHUB_UPBIT_SECRET_KEY": "your-secret-key"

          }
        }
      }
    }
    ```

    설정 후 Zed를 재시작하면 적용됩니다.

## Public API만 사용

인증 키 없이 Public API만 사용하려면 `env` 설정을 생략하세요:

```json
{
  "mcpServers": {
    "exhub-upbit": {
      "command": "npx",
      "args": ["@exhub/mcp", "--exchange", "upbit"]
    }
  }
}
```

시세 조회, 호가, 체결 내역 등 공개 데이터는 인증 없이 사용할 수 있습니다.
