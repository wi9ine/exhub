# 클라이언트 설정

사용하는 AI 클라이언트에 맞는 설정 방법을 확인하세요.

  <Tabs.Tab>
    ### Cursor
    
    [Cursor MCP documentation](https://docs.cursor.com/context/model-context-protocol)

    <Steps>
    #### 1. 설정 메뉴 진입 또는 파일 열기
    - **메뉴 활용 시:** `Settings` > `Features` > `MCP` > `+ Add new MCP server`
    - **파일 수정 시:** `~/.cursor/mcp.json` (글로벌) 또는 `.cursor/mcp.json` (프로젝트 내부) 파일 열기

    #### 2. 환경 변수 및 설정값 입력
    아래의 JSON 구문을 복사하여 설정 파일에 붙여넣습니다.
    ```json
    {
      "mcpServers": {
        "exhub-gopax": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "gopax"],
          "env": {
            "EXHUB_GOPAX_API_KEY": "your-api-key",
            "EXHUB_GOPAX_SECRET_KEY": "your-secret-key"
          }
        }
      }
    }
    ```

    #### 3. 변경사항 적용
    설정을 저장한 후, Cursor 에디터를 껐다가 다시 실행(Restart)하면 설정이 반영됩니다.
    </Steps>
  </Tabs.Tab>

  <Tabs.Tab>
    ### Claude Desktop
    
    [Claude Desktop MCP documentation](https://modelcontextprotocol.io/quickstart/user)

    <Steps>
    #### 1. 설정 파일 위치 열기
    사용 중인 운영체제에 맞춰 지정된 파일을 텍스트 에디터로 엽니다.
    - **macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`
    - **Windows:** `%APPDATA%\Claude\claude_desktop_config.json`

    #### 2. 환경 변수 및 설정값 입력
    아래 JSON 코문을 파일 안에 복사합니다.
    ```json
    {
      "mcpServers": {
        "exhub-gopax": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "gopax"],
          "env": {
            "EXHUB_GOPAX_API_KEY": "your-api-key",
            "EXHUB_GOPAX_SECRET_KEY": "your-secret-key"
          }
        }
      }
    }
    ```

    #### 3. 변경사항 적용
    파일을 수정한 후 Claude Desktop을 백그라운드에서 완전히 종료하고 켭니다.
    </Steps>
  </Tabs.Tab>

  <Tabs.Tab>
    ### VS Code (Cline / Roo Code)

    [Cline MCP documentation](https://github.com/cline/cline) | [Roo Code MCP documentation](https://docs.roocode.com/features/mcp/using-mcp-in-roo)

    <Callout type="info">
      VS Code 자체는 현재 내장 MCP 기능을 직접 지원하지 않으므로, **Cline**이나 **Roo Code**와 같은 인기 AI 편의 확장 프로그램을 필수로 이용해야 합니다.
    </Callout>

    <Steps>
    #### 1. 설정 메뉴 진입 또는 파일 열기
    - **Cline:** 사이드바 메뉴 톱니바퀴 > `MCP Servers` 탭 열기 (또는 `~/.vscode/mcp.json` 파일 수정)
    - **Roo Code:** 확장 프로그램 우측 상단 메뉴 > `MCP Servers` 클릭

    #### 2. 환경 변수 및 설정값 입력
    해당 설정 탭 혹은 JSON 파일 안에 아래 항목을 추가합니다.
    ```json
    {
      "mcpServers": {
        "exhub-gopax": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "gopax"],
          "env": {
            "EXHUB_GOPAX_API_KEY": "your-api-key",
            "EXHUB_GOPAX_SECRET_KEY": "your-secret-key"
          }
        }
      }
    }
    ```

    #### 3. 변경사항 적용
    파일을 저장하고 VS Code 창을 새로고침하여 활성화 여부를 확인하세요.
    </Steps>
  </Tabs.Tab>

    ### Claude Code
    
    [Claude Code MCP documentation](https://docs.anthropic.com/en/docs/agents-and-tools/mcp/install-mcp)

    <Steps>
    #### 1. CLI 터미널 연동 (선택)
    명령어를 통해 로컬 환경 명령줄에 곧바로 MCP 서버를 등록할 수 있습니다.
    ```bash
    claude mcp add exhub-gopax npx -y @exhub/mcp --exchange gopax
    ```

    #### 2. 구성 파일 직접 편집 (인증 키 필요 시)
    인증 키를 설정해야 한다면 위 명령어로 등록 후 생성된 `~/.claude.json` 파일을 열어 `env` 항목을 수동으로 추가하거나, 처음부터 직접 기입합니다.
    ```json
    {
      "mcpServers": {
        "exhub-gopax": {
          "command": "npx",
          "args": ["-y", "@exhub/mcp", "--exchange", "gopax"],
          "env": {
            "EXHUB_GOPAX_API_KEY": "your-api-key",
            "EXHUB_GOPAX_SECRET_KEY": "your-secret-key"
          }
        }
      }
    }
    ```
    </Steps>
  </Tabs.Tab>

    <Steps>

    #### 2. 환경 변수 및 설정값 입력
    설정 파일의 `mcpServers` 블록 내에 아래 내용을 추가합니다.
    ```json
    {
      "mcpServers": {
        "exhub-gopax": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "gopax"],
          "env": {
            "EXHUB_GOPAX_API_KEY": "your-api-key",
            "EXHUB_GOPAX_SECRET_KEY": "your-secret-key"
          }
        }
      }
    }
    ```

    #### 3. 변경사항 적용
    파일을 덮어쓰기 저장한 뒤, Windsurf 클라이언트를 재시작하면 완료됩니다.
    </Steps>

    ### Zed

    [Zed Context Server documentation](https://zed.dev/docs/assistant/context-servers)

    <Steps>
    #### 1. 설정 메뉴 진입
    상단 메뉴에서 `Zed` > `Settings` > `Open Settings` 를 클릭하거나 단축키(`Cmd + ,`)로 `settings.json`을 엽니다.

    #### 2. 환경 변수 및 설정값 입력
    `context_servers` 섹션을 찾아 코드를 추가하세요.
    ```json
    {
      "context_servers": {
        "exhub-gopax": {
          "command": "npx",
          "args": ["@exhub/mcp", "--exchange", "gopax"],
          "env": {
            "EXHUB_GOPAX_API_KEY": "your-api-key",
            "EXHUB_GOPAX_SECRET_KEY": "your-secret-key"
          }
        }
      }
    }
    ```

    #### 3. 변경사항 적용
    저장 후 설정 창을 닫으면 어시스턴트에서 도구를 즉시 활용할 수 있습니다.
    </Steps>
  </Tabs.Tab>

## 공통: Public API 전용 사용

시세 및 호가 조회 등 **인증(API Key/Secret_Key) 없이 공유 데이터를 호출**하려면 아래처럼 `env` 속성만 지워 등록해 주시면 어디서나 바로 사용할 수 있습니다.

```json
{
  "mcpServers": {
    "exhub-gopax": {
      "command": "npx",
      "args": ["@exhub/mcp", "--exchange", "gopax"]
    }
  }
}
```
