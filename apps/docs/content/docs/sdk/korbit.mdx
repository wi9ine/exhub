# Korbit

[Korbit](https://www.korbit.co.kr) 거래소 SDK 사용법입니다.

## 설치

```bash npm2yarn
npm install @exhub/korbit
```

## 클라이언트 생성

```typescript
import { createKorbitClient } from "@exhub/korbit";

const client = createKorbitClient();
```

## 인증 설정

Private API를 사용하려면 Korbit에서 API 키를 발급받으세요.

```typescript
const client = createKorbitClient({
  credentials: {
    apiKey: process.env.KORBIT_API_KEY!,
    secretKey: process.env.KORBIT_SECRET_KEY!,
  },
});
```

## Public API

| SDK 메서드 | 설명 | 공식 문서 |
|-----------|------|----------|
| `market.tickers(params)` | 현재가 조회 | [링크](https://docs.korbit.co.kr/#REST-get-_v2_tickers) |
| `market.orderbook(params)` | 호가 조회 | [링크](https://docs.korbit.co.kr/#REST-get-_v2_orderbook) |
| `market.trades(params)` | 최근 체결 내역 | [링크](https://docs.korbit.co.kr/#REST-get-_v2_trades) |
| `market.candles(params)` | 캔들스틱 조회 | [링크](https://docs.korbit.co.kr/#REST-get-_v2_candles) |
| `market.currencyPairs()` | 거래지원 목록 조회 | [링크](https://docs.korbit.co.kr/#REST-get-_v2_currencyPairs) |
| `market.tickSizePolicy(params)` | 호가 정책 조회 | [링크](https://docs.korbit.co.kr/#REST-get-_v2_tickSizePolicy) |
| `market.currencies()` | 가상자산 정보 조회 | [링크](https://docs.korbit.co.kr/#REST-get-_v2_currencies) |
| `market.time()` | 서버 시각 조회 | [링크](https://docs.korbit.co.kr/#REST-get-_v2_time) |

### 사용 예시

```typescript
// 현재가 조회
const tickers = await client.market.tickers();

// 호가 조회
const orderbook = await client.market.orderbook({ currency_pair: "btc_krw" });

// 캔들 조회
const candles = await client.market.candles({
  currency_pair: "btc_krw",
  interval: "1d",
});
```

## WebSocket API

### 클라이언트 생성

```typescript
import { createKorbitWsClient } from "@exhub/korbit";

const ws = createKorbitWsClient();
```

### 인증 설정 (Private 채널)

```typescript
const ws = createKorbitWsClient({
  credentials: {
    apiKey: process.env.KORBIT_API_KEY!,
    secretKey: process.env.KORBIT_SECRET_KEY!,
  },
});
```

### Public 채널

| SDK 메서드 | 설명 | 공식 문서 |
|-----------|------|----------|
| `subscribeTicker(onMessage)` | 현재가 실시간 수신 | [링크](https://docs.korbit.co.kr/#WS-method-subscribe_type-ticker) |
| `subscribeOrderbook(onMessage)` | 호가 실시간 수신 | [링크](https://docs.korbit.co.kr/#WS-method-subscribe_type-orderbook) |
| `subscribeTrade(onMessage)` | 체결 실시간 수신 | [링크](https://docs.korbit.co.kr/#WS-method-subscribe_type-trade) |

### Private 채널

<Callout type="warning">
  Private 채널은 인증 설정이 필요합니다. `credentials` 없이 호출하면 에러가 발생합니다.
</Callout>

| SDK 메서드 | 설명 | 공식 문서 |
|-----------|------|----------|
| `subscribeMyOrder(onMessage)` | 내 주문 실시간 수신 | [링크](https://docs.korbit.co.kr/#WS-method-subscribe_type-myOrder) |
| `subscribeMyTrade(onMessage)` | 내 체결 실시간 수신 | [링크](https://docs.korbit.co.kr/#WS-method-subscribe_type-myTrade) |
| `subscribeMyAsset(onMessage)` | 내 자산 실시간 수신 | [링크](https://docs.korbit.co.kr/#WS-method-subscribe_type-myAsset) |

### 사용 예시

```typescript
// 현재가 구독
const unsubscribe = ws.subscribeTicker((data) => {
  console.log(`${data.symbol}: ${data.data.close}`);
});

// 에러 처리
ws.onError((error) => console.error(error));

// 구독 해제
unsubscribe();

// 연결 종료
ws.close();
```
