# Coinone

[Coinone](https://coinone.co.kr) 거래소 SDK 사용법입니다.

## 설치

```bash npm2yarn
npm install @exhub/coinone
```

## 클라이언트 생성

```typescript
import { createCoinoneClient } from "@exhub/coinone";

const client = createCoinoneClient();
```

## 인증 설정

Private API를 사용하려면 Coinone에서 API 키를 발급받으세요.

```typescript
const client = createCoinoneClient({
  credentials: {
    accessToken: process.env.COINONE_ACCESS_TOKEN!,
    secretKey: process.env.COINONE_SECRET_KEY!,
  },
});
```

import { Callout } from "nextra/components";

<Callout type="info">
  Coinone은 다른 거래소와 달리 `accessToken`을 사용합니다. (`accessKey`가 아닙니다.)
</Callout>

## Public API

| SDK 메서드 | 설명 | 공식 문서 |
|-----------|------|----------|
| `market.getRangeUnit(quoteCurrency, targetCurrency)` | 개별 호가 단위 조회 | [링크](https://docs.coinone.co.kr/reference/range-unit) |
| `market.listMarkets(quoteCurrency)` | 전체 종목 정보 조회 | [링크](https://docs.coinone.co.kr/reference/markets) |
| `market.getMarket(quoteCurrency, targetCurrency)` | 개별 종목 정보 조회 | [링크](https://docs.coinone.co.kr/reference/market) |
| `market.getOrderbook(quoteCurrency, targetCurrency, params)` | 오더북 조회 | [링크](https://docs.coinone.co.kr/reference/orderbook) |
| `market.listTrades(quoteCurrency, targetCurrency, params)` | 최근 체결 주문 조회 | [링크](https://docs.coinone.co.kr/reference/recent-completed-orders) |
| `market.listTickers(quoteCurrency, params)` | 전체 티커 정보 조회 | [링크](https://docs.coinone.co.kr/reference/tickers) |
| `market.getTicker(quoteCurrency, targetCurrency, params)` | 개별 티커 정보 조회 | [링크](https://docs.coinone.co.kr/reference/ticker) |
| `market.listTickerUtc(quoteCurrency, params)` | 전체 티커 정보 조회 (UTC) | [링크](https://docs.coinone.co.kr/reference/utc-tickers) |
| `market.getTickerUtc(quoteCurrency, targetCurrency, params)` | 개별 티커 정보 조회 (UTC) | [링크](https://docs.coinone.co.kr/reference/utc-ticker) |
| `market.listCurrencies()` | 전체 가상자산 정보 조회 | [링크](https://docs.coinone.co.kr/reference/currencies) |
| `market.getCurrency(currency)` | 개별 가상자산 정보 조회 | [링크](https://docs.coinone.co.kr/reference/currency) |
| `market.getChart(quoteCurrency, targetCurrency, params)` | 캔들 차트 조회 | [링크](https://docs.coinone.co.kr/reference/chart) |

### 사용 예시

```typescript
// 현재가 조회
const tickers = await client.market.listTickers("KRW");

// 개별 티커 조회
const ticker = await client.market.getTicker("KRW", "BTC");

// 호가 조회
const orderbook = await client.market.getOrderbook("KRW", "BTC");
```

### 사용 예시

```typescript
// 잔고 조회
const balance = await client.account.listBalance();

// 주문 생성
const order = await client.orders.createOrder({
  target_currency: "BTC",
  quote_currency: "KRW",
  type: "limit",
  side: "buy",
  price: "50000000",
  qty: "0.001",
});

// 주문 취소
await client.orders.cancelOrder({ order_id: "order-id" });
```
