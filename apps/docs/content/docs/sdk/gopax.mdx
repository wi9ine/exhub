# Gopax

[Gopax](https://www.gopax.co.kr) 거래소 SDK 사용법입니다.

## 설치

```bash npm2yarn
npm install @exhub/gopax
```

## 클라이언트 생성

```typescript
import { createGopaxClient } from "@exhub/gopax";

const client = createGopaxClient();
```

## 인증 설정

Private API를 사용하려면 Gopax에서 API 키를 발급받으세요.

```typescript
const client = createGopaxClient({
  credentials: {
    apiKey: process.env.GOPAX_API_KEY!,
    secretKey: process.env.GOPAX_SECRET_KEY!,
  },
});
```

## Public API

| SDK 메서드 | 설명 | 공식 문서 |
|-----------|------|----------|
| `market.assets()` | 자산 목록 조회 | [링크](https://gopax.github.io/API/index.html#f55085599b) |
| `market.tradingPairs()` | 거래쌍 목록 조회 | [링크](https://gopax.github.io/API/index.html#bd57884325) |
| `market.priceTickSize(tradingPair)` | 가격 틱 사이즈 조회 | [링크](https://gopax.github.io/API/index.html#8cbaa14a64) |
| `market.ticker(tradingPair)` | 티커 조회 | [링크](https://gopax.github.io/API/index.html#c63d14aba7) |
| `market.tickers()` | 전체 티커 조회 | [링크](https://gopax.github.io/API/index.html#5273f952f5) |
| `market.orderbook(tradingPair, params)` | 오더북 조회 | [링크](https://gopax.github.io/API/index.html#6ad081727e) |
| `market.trades(tradingPair, params)` | 체결 기록 조회 | [링크](https://gopax.github.io/API/index.html#c3c0208b04-2) |
| `market.stats(tradingPair)` | 최근 24시간 통계 조회 | [링크](https://gopax.github.io/API/index.html#24) |
| `market.allStats()` | 최근 24시간 통계 조회 (모든 거래쌍) | [링크](https://gopax.github.io/API/index.html#24-2) |
| `market.candles(tradingPair, params)` | 차트 데이터 조회 | [링크](https://gopax.github.io/API/index.html#10f99c8763) |
| `market.cautions(params)` | 투자유의 정보 조회 | [링크](https://gopax.github.io/API/index.html#e4dbb6ad96) |
| `market.time()` | 서버 시간 조회 | [링크](https://gopax.github.io/API/index.html#31180a60f1) |
| `market.notices(params)` | 공지사항 조회 | [링크](https://gopax.github.io/API/index.html#53976d821f) |

### 사용 예시

```typescript
// 현재가 조회
const ticker = await client.market.ticker("BTC-KRW");

// 모든 티커 조회
const allTickers = await client.market.tickers();

// 호가 조회
const orderbook = await client.market.orderbook("BTC-KRW");
```

### 사용 예시

```typescript
// 잔고 조회
const balances = await client.account.listBalances();

// 주문 생성
const order = await client.orders.createOrder({
  trading_pair: "BTC-KRW",
  side: "buy",
  type: "limit",
  price: "50000000",
  amount: "0.001",
});

// 주문 취소
await client.orders.cancelOrder("order-id");
```
