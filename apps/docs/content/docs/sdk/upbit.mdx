# Upbit

[Upbit](https://upbit.com) 거래소 SDK 사용법입니다.

## 설치

```bash npm2yarn
npm install @exhub/upbit
```

## 클라이언트 생성

```typescript
import { createUpbitClient } from "@exhub/upbit";

const client = createUpbitClient();
```

## 인증 설정

Private API를 사용하려면 [Upbit Open API](https://upbit.com/mypage/open_api_management)에서 API 키를 발급받으세요.

```typescript
const client = createUpbitClient({
  credentials: {
    accessKey: process.env.UPBIT_ACCESS_KEY!,
    secretKey: process.env.UPBIT_SECRET_KEY!,
  },
});
```

### 사용 예시

```typescript
// 현재가 조회
const tickers = await client.tickers.listTickers({ markets: "KRW-BTC" });

// 일봉 조회
const candles = await client.candles.getDayCandles({ market: "KRW-BTC", count: 10 });

// 분봉 조회 (1, 3, 5, 10, 15, 30, 60, 240분)
const minuteCandles = await client.candles.getMinuteCandles(1, {
  market: "KRW-BTC",
  count: 10,
});
```

### 사용 예시

```typescript
// 잔고 조회
const balance = await client.assets.listBalance();

// 지정가 매수 주문
const order = await client.orders.createOrder({
  market: "KRW-BTC",
  side: "bid",
  ord_type: "limit",
  price: "50000000",
  volume: "0.001",
});

// 주문 취소
await client.orders.cancelOrder({ uuid: order.uuid });
```
