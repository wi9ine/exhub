# Bithumb

[Bithumb](https://www.bithumb.com) 거래소 SDK 사용법입니다.

## 설치

```bash npm2yarn
npm install @exhub/bithumb
```

## 클라이언트 생성

```typescript
import { createBithumbClient } from "@exhub/bithumb";

const client = createBithumbClient();
```

## 인증 설정

Private API를 사용하려면 Bithumb에서 API 키를 발급받으세요.

```typescript
const client = createBithumbClient({
  credentials: {
    apiKey: process.env.BITHUMB_API_KEY!,
    secretKey: process.env.BITHUMB_SECRET_KEY!,
  },
});
```

### 사용 예시

```typescript

// 일봉 조회
const candles = await client.candles.getDayCandles({ market: "KRW-BTC", count: 10 });
```
